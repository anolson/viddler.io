== viddler.io

A Rails (3.2) to demo some features of Viddler[http://developers.viddler.com/]

== Getting started

Clone the source.

  $ git clone https://github.com/anolson/viddler.io
  $ cd viddler.io
  $ bundle
  
Configure your Viddler API key.

  $ export VIDDLER_API_KEY=<your api key>
  
Start the server.
  
  $ bundle exec rails s
  
Run the tests.

  $ bundle exec guard
